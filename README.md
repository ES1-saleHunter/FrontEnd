# SaleHunter


## Requisitos

- Node 16.19
- Vue 3.x  
- Tailwind 
- Element

## Instalação

1. Clone o repositório em sua máquina local:

   `git clone https://github.com/eugeogeo/api-nobanco.git`

2. Instalar as dependências:

   `yarn install`

3. Para rodar a primeira vez

   `yarn build`

   `yarn serve`
